# Dice / Dados

#### English
**Simple application created with flutter to roll a dice from 1 to 6. Whenever you touch the "Tira el dado" text, it will
roll the dice for you and give you a random number from 1 to 6. **

### To-Do List to improve the app

- Make multiple dice types available
- Make an intro animation
- Animate dices and add a sound to it when the button is pressed
- Make an intro page with a language selector to translate the app to multiple languages.
